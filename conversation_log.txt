Claude:
    GitHub has two relevant concepts here:

    Collaborators (people with push access) are added via repository settings.
    Contributors are automatically tracked by GitHub based on who has made
    commits — you can't manually add them.

    Adding a Collaborator:
    1. Go to https://github.com/sohaamir/claude_code_demo
    2. Click Settings (top tab)
    3. Click Collaborators in the left sidebar (under "Access")
    4. Click Add people
    5. Search by GitHub username and invite them

    Regarding "claude" as a Contributor:
    Claude (the AI) is already appearing as a contributor in your commit
    history via the Co-Authored-By tags added to commits. You can verify
    this by checking the Contributors graph on GitHub:
    https://github.com/sohaamir/claude_code_demo/graphs/contributors

    GitHub recognises Co-Authored-By lines in commit messages and attributes
    those commits to the specified authors, so Claude's contributions are
    already being tracked there.
